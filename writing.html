➡️ Writing

Multimedia notes and collaborative docs

  • Google Docs (💵): Obligatory.
  • Jupyter Notebook (📍🥨✨): Can render data visualizations. High customization. Requires coding knowledge.
  • Etherpad (📍🥨✨): Open source. Runs on local server, can share with computer IP.
  • Yopad (📍🧭✨): Built with etherpad. Ready to use.
  • Notion (📍💵): Desktop and mobile application. Has templates. Collaborative.
  • Evernote (📍💵): Multi-platform. Note-taking application.
  • Simple Note (✨): Similar to Evernote. Cross-platform.
  • Outline (✨): Usually used for internal documentation.
  • Coda (💰): Desktop application. Has templates. Collaborative.
  • Notepad++ (✨): Works only on Windows, though there are ways to get around this.
  • Creately (💰): Collaborative. Can visualize computer networks, charts.
  • Scripto (💵): Used for screen-writing, but has collaborative features that can be used for writing documents. Integrates video.

Note-taking

  • Libre Office (✨📍): Office suite, similar to Microsoft office (but non-proprietary).
  • Roam Research (💰): Note-taking software. Uses Markdown (kinda). They have a scholars program.
  • Foam (✨📍): Inspired by Roam but open source.
  • Ulysses (💰): Writing app. Phone and desktop app.
  • Bear (💵): Apple products only. Write in markup.
  • Dokuwiki (✨): Wiki software. Saves as txt. May require some coding skills.
  • Org-roam (✨): Plain-text knowledge management. May require some coding skills.
  • Digital Garden (📍✨): List of digital gardening tools & templates. Notes are public, cross-linked & networked.
  • Hyperdraft (✨): Networked notes. Works offline. Web-based.
  • Zettlr (✨🧭): Multi-platform. Markdown editor for scholarly writing. Zotero integration.
  • Obsidian (✨🧭): Multi-platform (desktop and mobile) networked markdown knowledge base with plugins and publish.

Concept Mapping

  • Miro (💵 🧭): Collaborative visual brainstorming. Student discount. No coding needed.
  • TiddlyMap (✨): Combines wikis and concept-maps.
  • Kumu (💵): Lots of different mapping options.
  • Coggle (💵): Visual. No coding needed.

Miscellaneous

  • Perligo (💵): Peer and group feedback on writing.
  • Connected Papers: Uses network analysis to visualize academic articles as a network
