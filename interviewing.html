➡️ Interviewing

Transcription

  • oTranscribe (📍✨): Web-based. Youtube video or audio file. Self-transcription.
  • Express Scribe: Available for mac. Designed for typists.
  • Parlatype (✨): Minimal audio player.
  • Scripto (✨): Crowdsourced transcription.
  • Transcribe by Wreally (💰): Manual and automatic transcription packages.
  • Happy Scribe (💵): Automated. Version available for students. Multilingual support.
  • Scribie (💰): Paid service.
  • Rev (💰): Transcription service. Can add captions to videos.
  • Take Note (💰): Transcription service.
  • Otter.ai (💵): Separates by time stamps. Basic version available. Requires account.
  • Way With Words (💰): Paid service. Manual transcription.
  • Trint (💰): Free trial. Can search and annotate transcription.

Coding and Analysis

  • Nvivo (💰): Automated transcription. Analysis tools. Desktop software. Free trial available.
  • Atlas.ti (💰): Free trial available. Desktop and phone support. Allows for sentiment analysis.
  • Delve Tool (📍💰): Annotation tool. Free trial available.
  • MaxQDA (💰): Discounted license for students.
  • Quirkos (💰): Free trial.
  • Dedoose (💰): Web app.
  • Provalis (💰): Software suite.
  • WebQDA (💰): Free trial.
  • Hyper Research (💵): Keyword mapping, mind mapping.
  • Qualcoder (✨): Desktop app.

Translation

  • Google Translate (💵): Obligatory.
  • DeepL (💵): Machine-assisted. App available.
  • Translators Café: Forum of professional translators.
  • Text United (💰): Free trial.

Miscellaneous

  • Oral History as Data: Can analyse and publish oral history transcripts, color-coded by type.
