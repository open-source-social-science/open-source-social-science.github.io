➡️ Data collection

Collecting Links

  • Are.na (✨📍): "Mood board"-like. Has mobile app and browser extension. No desktop app.
  • Pocket (💵): Desktop app and mobile app.
  • Raindrop.io (💵📍✨): Desktop and mobile app.
  • Tab Session Manager (✨📍): Browser extension. Available for Chrome.
  • Wallabag (✨): Can highlight directly on saved pages. No app.
  • ArchiveBox.io (🥨✨): Stores all data locally. Can read lists from other archiving apps.

Web-scraping

  • Webscraper.io (💵📍): Chrome extension. No coding
  • Parsehub: Chrome extension. No coding.
  • Parsehub (💵): Desktop app. No coding.
  • Scrapy (✨): Some coding required.
  • BeautifulSoup (🥨📍✨): Python library. Good documentation.
  • Octoparse (💰): Free demo.
  • Google News Scraper (🧭): Requires Firefox.

Word Counting

  • Word Counter (✨): No coding needed. Generates list of frequently-used words.
  • Tag Cloud Generator: Similar function.

Miscellaneous

  • Screennotate (📍): Automatically grabs texts from screenshots, along with the URL and the title of the place where you took the screenshot (where possible).
  • Grain (💵): Allows userts to record, transcribes, and share highlights of Zoom calls.
  • Breadboard (🥨): For online network experiments.
  • Lentil (✨): Harvested photos from Instagram based on hashtags. No longer supported.
  • Voyant Tools (✨): Environment for analysing digital texts.
  • Juxta (✨): tool for comparing and collating multiple witnesses to a single textual work. Desktop app.
  • Internet Archive Wayback Machine Link Ripper (✨): Grabs links from archive.
  • 4CAT - Capture and Analysis Toolkit (✨): Create datasets from web forums.
  • Minet (✨): By SciencePo MédiaLab. Mass web-mining software. Requires command line.
  • Convertio (💵): Converts html files to doc files. Chrome extension and web app.
  • Connected Papers: Visual tool for exploring connected papers and scholarship.
  • Dribdrat (✨): Organize hackathons to crowdsource ideas, patches & prototypes.

Resources

  • Doing Fieldwork in a Pandemic: Collective google-doc on methods, approaches, and ethical concerns during a pandemic.
  • Buildling LLTDM: Legal Literacies for Text Data Mining.
  • Social Feed Manager: Tools to help researchers and archivists to build social media collections.
  • Digital Methods Initiatives: Has many tools for researchers. Runs winter and summer school.
  • MédiaLab: Research lab at SciencePo in Paris, France. Has list of tools.
  • Toolkit: Digital Ethics: Collected by Shannon Mattern.
  • AoIR Ethics: Association of Internet Researchers’ ethics resources.
  • Toolkit: Digital Ethnography: Collected by Shannon Mattern.
  • Critical Data Studies Resources: Reading lists and organizations. By Angèle Christin.
  • Digital Ethnography Collective Reading list (✨): LSE's Digital Ethnography Collective Reading List, hosted by Zoë Glatt.
