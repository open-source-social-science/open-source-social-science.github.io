➡️ Visual

Photography and Graphic Design

  • Canva (💵📍): Edit photos, create and design documents. Templates available.
  • DesignBold (💵): Alternative to canva.
  • Gimp (✨): Image editor.

Modeling

  • Blender (✨): 3D rendering.
  • Inkscape (✨): Vector graphics editor.
  • SVG Edit (✨): SVG editor in browswer.
  • Unity (💵✨): 3D rendering software. Selectively open source.

Charts and Data Visualization

  • Datawrapper (✨💵🧭📍): Easy upload/import system for CSV files. Very user-friendly for non-coders.
  • Tableau (💰): Free trial available.
  • Infogram: Web-based.
  • HighCharts (📍): Javascript
  • HighCharts (💰): Free for non-commercial use. Basic Javscript coding required.
  • D3.js (✨🥨): Javascript library. Lots of customization.
  • Local Focus (🧭): Can visualize maps and charts. No coding needed.
  • Raw Graphs (✨🧭): Upload data directly on the web, variety of viz options.
  • Zettlr Charter (✨🧭): Open Source alternative to Datawrapper. Allows upload of CSV/TSV files and ad-hoc data curation.

Mapping

  • Story Maps (💰): From ArcGIS.
  • StoryMap JS (✨): Open source version of the one from ArcGIS. No coding required.
  • Leaflet (✨📍): Requires knowledge of Javascript.
  • MapBox (💵📍): Customizable map. Has edit features for data.

Miscellaneous

  • Listify (✨): Turns spreadsheet into searchable visual list.
  • Databin (✨): Share tabular data.
